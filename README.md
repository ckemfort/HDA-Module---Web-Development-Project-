# Project - Purrrfect Match
Welcome to my university project Purrrfect Match which was built as part of the module "Project Web Entwicklung". 
Main tools: SQLite, Python, Flask, and Bootstrap

## Description
This is a gallery of cat breeds with images, descriptions and traits. The database was compiled using two main data sources. TheCatAPI and WikiData.

# Goal
Competencies for developing web- and database-based software in a systematic and user-oriented manner for typical information science application scenarios. 
Implement an end-to-end data pipeline using at least two different data sources.
Expan on the knowledge acquired in the corresponding basic courses on semantics, information design, and software engineering.

## Installation
The dependencies can be seen in the .toml file 
Run scripts/init_db.py to initialise the SQLite database
Run the /scripts/seed_db.py to seed the Database
Run app/app.py to start the flask server
